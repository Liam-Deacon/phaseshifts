# Reporting Issues and Getting Help

We welcome bug reports, enhancement requests, and general feedback.

## Bug reports and enhancement requests

- Open a GitHub issue: https://github.com/Liam-Deacon/phaseshifts/issues
- Use the issue template and include reproduction steps, logs, and version
  information when possible.

We aim to acknowledge new reports within 14 days when possible. GitHub issues
serve as the public archive for reports and responses.

## Discussion and questions

- Use GitHub Discussions: https://github.com/Liam-Deacon/phaseshifts/discussions
- Discussions are searchable and addressable by URL, and are open to new
  participants.

## Security issues

Please do not open public issues for security reports. See SECURITY.md for the
private reporting process.

## Language

Project communication is in English, and we can accept bug reports and feedback
in English.
